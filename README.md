# AccApp

スマートフォンの加速度センサーを利用して、スマホを振った回数をカウントするWebアプリケーションです。
GitHub Pages上で動作確認を行いながらテスト駆動で開発しています。

## デモ

https://srn72893.github.io/AccApp/

※スマートフォンのブラウザでアクセスしてください。

---

## 機能

- 加速度センサーの取得
- センサー値のリアルタイム表示
- スマートフォンを振った回数のカウント
- シェイク検出（閾値 + クールダウン）

---

## 使用技術

- HTML
- JavaScript

---

## 使い方

1. スマートフォンでデモページにアクセス
2. startボタンを押す（センサー値のリアルタイム取得）
3. count startボタンを押す（振った回数のカウント）
4. 適宜センサー使用の許可を与える
5. スマートフォンを振るとカウントが増える

---

## 動作環境

- iOS Safari
- Android Chrome

※PCではセンサーが利用できないため動作しません。
